## _IS_PATH_FOR_GPS_ON_ROAD

```cpp
// 0xF47A1EB2A538A3A3
bool _IS_PATH_FOR_GPS_ON_ROAD();
```

Checks if the GPS route to the waypoint is navigable along a road. If a route exists but there is no valid road path, this function returns false.


## Return value
