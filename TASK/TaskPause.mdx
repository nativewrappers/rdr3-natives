## TASK_PAUSE

```cpp
// 0xE73A266DB0CA9042
void TASK_PAUSE(Ped ped, int ms);
```

This tasks the ped to do nothing for the specified amount of milliseconds.
This is useful if you want to add a delay between tasks when using a sequence task.
