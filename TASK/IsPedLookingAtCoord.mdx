## _IS_PED_LOOKING_AT_COORD

```cpp
// 0x508F5053E3F6F0C4
bool _IS_PED_LOOKING_AT_COORD(Ped ped, Vector3 coord, float radius);
```

hecks if the ped camera is directed at the given coord within the specified radius.

https://www.youtube.com/watch?v=iLHXrWVvq_c

## Parameters
* **ped**:
* **coord**: The coord to check if they're looking at
* **radius**: The radius of the frustum
