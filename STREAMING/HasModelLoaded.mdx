## HAS_MODEL_LOADED

```cpp
// 0x1283B8B89DD5D1B6
BOOL HAS_MODEL_LOADED(Hash model);
```

Checks if the specified model has loaded into memory.

## Parameters
* **model**: The model to load, you should check that its valid with [native: IS_MODEL_IN_CDIMAGE]

## Return value
Returns `true` if the specified model has been loaded into streaming memory, `false` otherwise.
